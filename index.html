🛰️ Conectar con CDSE

Usamos la API gratuita de Copernicus Data Space. Registrate en dataspace.copernicus.eu (gratis) y usá el mismo usuario y contraseña aquí. El token se obtiene automáticamente y se renueva cada sesión.

🔒 Las credenciales se guardan en localStorage de tu navegador.
💾 Para que persistan entre dispositivos/sesiones, editá el archivo credentials.js con tu email y contraseña CDSE (y agregalo a .gitignore).
